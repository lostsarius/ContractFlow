# Icons erstellen

Um echte PWA-Icons zu erstellen, verwenden Sie einen Icon-Generator wie:

## Online Tools:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

## Manuell:

Erstellen Sie zwei PNG-Dateien:
- `icon-192x192.png` (192x192 Pixel)
- `icon-512x512.png` (512x512 Pixel)

Und platzieren Sie sie im `public/` Ordner.

## Temporäre Lösung:

Die App verwendet das SVG-Icon (`icon.svg`) als Fallback.
